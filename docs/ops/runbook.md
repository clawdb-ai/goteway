# 运行与排障手册

日期: 2026-03-17

## 1. 启动原则

- 默认绑定 `127.0.0.1`
- 远程访问启用 token + TLS
- 启动前检查配置、密钥、插件目录

## 2. 健康检查

- `GET /health` 返回 `status=ok`
- 关键指标:
  - ws_active_connections
  - ws_send_queue_depth
  - req_latency_ms
  - plugin_call_timeout_total

## 3. 常见故障

### 3.1 握手失败率升高
- 检查协议版本窗口
- 检查认证配置与 token 过期
- 查看网关错误码分布

### 3.2 广播积压
- 检查慢消费者连接列表
- 临时调低广播频率或启用降级
- 提升发送队列阈值需谨慎

### 3.3 插件调用超时
- 检查插件运行时负载
- 调整单插件并发和超时
- 触发熔断后观察恢复

## 4. 回滚流程

1. 切流开关回退到官方 Gateway
2. 保留会话与幂等状态数据
3. 验证握手成功率恢复
4. 归档故障窗口日志与 trace

## 5. 变更审计

- 认证策略变更
- 插件配置变更
- 路由规则变更
- 限流策略变更
